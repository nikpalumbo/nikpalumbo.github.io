# Intelligent Growth System - Pricing Rules

## Overview
The Intelligent Growth System pricing ranges from **$5,000 to $15,000** base, with additional costs based on complexity factors.

## Base Price
- **Starting Price**: $5,000 (includes basic setup with 2 marketing channels)

## Pricing Factors & Weights

### 1. Company Stage (30% weight) - Based on Experience Level
- **Just Starting**: +$0 (baseline)
  - Haven't used a growth system before
  - No existing automation
  - Need full setup and training
- **Ready to Scale**: +$2,500
  - Have some growth systems but need to scale them
  - Some automation already in place
  - Need to expand and optimize existing processes
- **Enterprise**: +$5,000
  - Complex existing systems that need integration
  - Multiple systems already in place
  - Need advanced integration and enterprise-level features

### 2. Marketing Channels (25% weight)
- **Base**: $5,000 (includes 2 channels)
- **Additional channels**: +$500 each
- **Available channels**: Email Inbound, Email Outbound, SEO, LinkedIn, Website Forms, Ads Integration, Social Media
- **Maximum channels**: 7

### 3. CRM Integration Complexity (25% weight)

#### Easy Integration (+$0)
- HubSpot
- Pipedrive
- Freshworks CRM
- Insightly
- Nimble

#### Medium Integration (+$1,000)
- Zoho CRM
- vTrek

#### Difficult Integration (+$2,000)
- Salesforce
- Microsoft Dynamics
- SugarCRM

#### Additional CRM Fees
- **Multiple CRMs**: +$1,000 complexity fee
- **No CRM**: +$500 (setup from scratch)

### 4. Optimization Frequency (10% weight)
- **Monthly**: +$1,000
- **Quarterly**: +$500
- **Bi-weekly**: +$1,500

### 5. Delivery Timeline (10% weight)
- **2 weeks**: +$2,000 (rush fee)
- **1 month**: +$0 (standard)
- **2 months**: +$0 (standard)

- **Maximum recommended price**: $20,000
- **Complexity discounts**: Available for long-term contracts
- **Custom integrations**: Additional costs may apply
- **Rush delivery**: Only available for projects under $15,000 base
- **Experience-based pricing**: Reflects actual work complexity and training needs
